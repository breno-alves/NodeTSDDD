# NodeDDD API

NodeDDD's API

## Requirements

- Node v14
- NPM
- Docker
- Docker-compose

## Running application with Makefile

```
make setup

make migration

make start
```

## Version

    Node.js v14.15.1

## Install dependecies

    npm i

## Run Api in development

    npm run dev:server

## Run migrations

    npm run typeorm migration:run

---
